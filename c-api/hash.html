PyHash API¶

另請參閱 PyTypeObject.tp_hash 成員。

type Py_hash_t¶

雜湊值型別：有符號整數。

在 3.2 版被加入.

type Py_uhash_t¶

雜湊值型別：無符號整數。

在 3.2 版被加入.

type PyHash_FuncDef¶

PyHash_GetFuncDef() 所使用的雜湊函式定義。

const char *name¶

雜湊函式名稱（UTF-8 編碼字串）。

const int hash_bits¶

雜湊值的內部大小（以位元為單位）。

const int seed_bits¶

Seed 輸入的大小（以位元為單位）。

在 3.4 版被加入.

PyHash_FuncDef *PyHash_GetFuncDef(void)¶

取得雜湊函式定義。

也參考

PEP 456「安全且可交替使用的雜湊演算法 (Secure and interchangeable hash algorithm)」。

在 3.4 版被加入.
