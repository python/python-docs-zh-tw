工具¶

本章中的函式可用來執行各種工具任務，包括幫助 C 程式碼提升跨平臺可攜性 (portable)、在 C 中使用 Python module（模組）、以及剖析函式引數並基於 C 中的值來構建 Python 中的值等。

  • 作業系統工具
  • 系統函式
  • 行程（Process）控制
  • 匯入模組
  • Data marshalling support
  • Parsing arguments and building values
    • Parsing arguments
      • Strings and buffers
      • 數字
      • 其他物件
      • API 函式
    • Building values
  • 字串轉換與格式化
  • Reflection
  • Codec registry and support functions
    • Codec lookup API
    • Registry API for Unicode encoding error handlers
